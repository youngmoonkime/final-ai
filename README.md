# AI 제주 여행 가이드 (AI Jeju Travel Guide)

AI 제주 여행 가이드는 Google Gemini API를 활용하여 사용자의 여행 스타일에 맞는 맞춤형 제주도 여행지를 추천해주는 웹 애플리케이션입니다. AI가 생성하는 아름다운 이미지와 상세한 정보를 통해 최고의 제주 여행을 계획해보세요.

## ✨ 주요 기능

*   **✈️ 개인 맞춤형 추천**: 사용자의 연령대, 여행 스타일, 여행 유형에 따라 Gemini AI가 최적의 여행지를 추천합니다.
*   **🖼️ AI 이미지 생성**: 각 추천 장소의 특성에 맞는 고유하고 생생한 이미지를 AI가 실시간으로 생성합니다.
*   **ℹ️ 상세 정보 제공**: AI가 요약한 리뷰, 운영 시간, 주요 메뉴 등 방문에 필요한 모든 정보를 한눈에 볼 수 있습니다.
*   **⭐ 북마크 기능**: 마음에 드는 장소를 저장하고, 북마크한 장소만 모아볼 수 있습니다.
*   **🗺️ 외부 서비스 연동**: Google 지도 길찾기와 인스타그램 해시태그 검색으로 바로 연결하여 더 많은 정보를 얻을 수 있습니다.
*   **🔄 동적 필터링**: 추천 목록 화면에서 언제든지 여행 스타일을 변경하고 새로운 추천을 받을 수 있습니다.

## 🛠️ 기술 스택

*   **프론트엔드**: React, TypeScript, Tailwind CSS
*   **AI 모델**: Google Gemini API
    *   `gemini-2.5-flash`: 텍스트 기반 여행지 추천
    *   `gemini-2.5-flash-image`: 각 장소에 맞는 이미지 생성

## 🚀 작동 방식

1.  **여행 스타일 설정**: 사용자가 앱의 첫 화면에서 자신의 여행 스타일(연령대, 스타일, 유형)을 선택합니다.
2.  **AI 추천 요청**: 선택된 스타일을 기반으로 Gemini API (`gemini-2.5-flash`)에 맞춤 여행지 추천을 요청합니다.
3.  **이미지 생성**: AI가 추천한 각 장소의 이름과 설명을 바탕으로 Gemini API (`gemini-2.5-flash-image`)를 통해 고유한 이미지를 생성합니다.
4.  **결과 표시**: 생성된 이미지와 함께 AI 추천 목록을 사용자에게 보여줍니다.
5.  **앱 기능 활용**: 사용자는 목록을 탐색하고, 상세 정보를 확인하며, 북마크, 지도/인스타그램 연동 등 다양한 기능을 활용하여 여행을 계획합니다.

## 📂 파일 구조

-   `index.html`: 애플리케이션의 기본 HTML 파일.
-   `index.tsx`: React 앱을 DOM에 렌더링하는 진입점.
-   `App.tsx`: 애플리케이션의 메인 컴포넌트. 뷰 전환, 상태 관리 등 핵심 로직을 담당.
-   `services/geminiService.ts`: Google Gemini API와 통신하여 추천 데이터와 이미지를 가져오는 서비스.
-   `types.ts`: 앱 전체에서 사용되는 TypeScript 타입 정의.
-   `constants.ts`: 여행 스타일 옵션, 초기 데이터 등 앱에서 사용되는 상수 데이터.
-   `metadata.json`: 앱의 이름, 설명 등 메타데이터.
-   `README.md`: 프로젝트 개요 및 설명 (현재 파일).
